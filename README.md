# 🏆 UIDAI Data Hackathon 2026 Finalist

This repository contains my **Aadhaar Enrolment Intelligence System**, developed as part of the **UIDAI Data Hackathon 2026**, where my team was recognized as a **Finalist**.

---

# 📖 Overview

The Aadhaar Enrolment Intelligence System transforms raw Aadhaar enrolment datasets into actionable insights using Data Science, Machine Learning, Streamlit and Power BI.

## ✨ Features

- Data Cleaning
- Feature Engineering
- Interactive Dashboards
- Power BI Reports
- State & District Analytics
- Heatmaps
- KPI Cards
- Forecasting
- Linear Regression
- ARIMA
- District Clustering
- Anomaly Detection
- Government Decision Support
